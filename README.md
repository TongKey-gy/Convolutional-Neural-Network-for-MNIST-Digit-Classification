 MNIST 손글씨 숫자 분류 (CNN)
 <Convolutional Neural Network for MNIST Digit Classification>
Keras와 CNN(Convolutional Neural Network)을 이용하여 MNIST 손글씨 숫자(0~9)를 분류하는 딥러닝 모델 구현 프로젝트입니다.
🚀 사용 기술 (Tech Stack)

Language: Python
Framework: TensorFlow / Keras
Library: NumPy
Model: CNN (Convolutional Neural Network) + Dense Layer


<요약>

CNN을 이용해 MNIST 이미지의 패턴을 추출하고 분류함
5 Epoch 학습 후 약 99% 이상의 높은 정확도 달성
딥러닝 기본 구조(CNN → Flatten → Dense)를 이해.
